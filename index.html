Traffic Light Simulator

Welcome to the Traffic Light Simulator! This application allows you to simulate the behavior of traffic lights at an intersection. You can control the traffic lights and observe how they change over time.

Its Features

  • Simulate traffic light cycles (red, yellow, green)
  • Control the duration of each light
  • Visual representation of the traffic lights

How to Use

Click the "Start Simulation" button below to start.For each light you can adjust the duration using the provided controls. The traffic lights will change according to the set durations.
